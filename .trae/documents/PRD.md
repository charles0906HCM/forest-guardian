## 1. 产品概述

小学生打卡任务应用「森林小卫士」—— 一款帮助孩子养成良好时间管理习惯的任务管理工具。通过番茄钟专注、四象限优先级分类和星愿币激励机制，让孩子在森林童话氛围中学会规划时间、完成任务、收获奖励。

- **目标用户**：6-12 岁小学生及其家长
- **核心价值**：用游戏化机制培养孩子的时间管理能力与专注力，让好习惯养成变得有趣
- **设计理念**：大自然森林系半透明毛玻璃风格，营造沉浸式森林探险体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|---------|----------|
| 小学生 | 直接使用 | 创建/修改任务、启动番茄钟、获得星愿币、兑换奖品 |
| 家长 | 协助使用 | 帮助设置任务、管理奖品清单、查看孩子完成情况 |

### 2.2 功能模块

1. **任务清单页**：四象限任务分类、任务列表、添加/修改任务、重复任务设置、星愿币与番茄钟配置
2. **番茄钟页**：番茄计时器、任务关联、专注进度、完成奖励
3. **星愿币页**：星愿币统计、好/坏习惯管理、奖品兑换清单
4. **成绩单页**：各科考试成绩录入、考试记录列表、按科目/日期筛选
5. **成绩分析页**：成绩趋势图表、科目对比、进步退步分析、目标达成情况

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 任务清单页 | 四象限清单 | 按重要紧急/重要不紧急/不重要紧急/不重要不紧急四象限展示任务卡片 |
| 任务清单页 | 任务列表 | 按日期筛选任务,展示任务名称、时间、星愿币、番茄钟时长,已完成任务名称画横线 |

| 番茄钟页 | 专注控制 | 开始/暂停/重置按钮,专注完成后自动发放星愿币奖励 |

| 成绩分析页 | 趋势图表 | 各科目成绩折线趋势图、得分率变化、平均分参考线 |

## 3. 核心流程

**主流程**：用户打开应用 → 查看今日任务清单 → 点击任务名称启动番茄钟 → 专注完成 → 任务自动标记完成(名称画横线) → 获得星愿币 → 在星愿币页查看统计或兑换奖品。

**任务创建流程**：点击新建任务 → 填写任务名称 → 选择开始结束日期时间 → 设置重复选项(每日/每周) → 设置重复结束日期 → 设置星愿币数量 → 选择番茄钟时长(5/10/15/25 分钟或自定义) → 选择所属象限 → 保存任务。

**星愿币流转流程**：完成任务/好习惯 → 星愿币增加 → 在奖品清单中选择奖品 → 点击兑换 → 星愿币扣减 → 兑换记录保存。

**成绩录入流程**：进入成绩单页 → 点击录入成绩 → 选择考试名称与日期 → 逐科录入科目、得分、满分、排名 → 设置目标分数 → 保存 → 系统自动判断是否达成目标,达成则发放星愿币奖励 → 在成绩分析页查看趋势。

## 4. 用户界面设计

### 4.1 设计风格

**大自然森林系半透明毛玻璃风格**——营造清晨森林的宁静与生机。

- **主色调**：
  - 森林深绿 `#2D5F3F` (主色)
  - 苔藓嫩绿 `#7FB069` (辅助色)
  - 晨雾白 `#F4F7F5` (背景)
  - 阳光金黄 `#F4B942` (星愿币/强调色)
  - 树皮棕 `#8B6F47` (文字次要)
- **毛玻璃效果**：`backdrop-blur(20px)` + 半透明白色背景 `rgba(255,255,255,0.4)`,卡片边缘 `1px solid rgba(255,255,255,0.6)` 内发光
- **背景**：分层森林渐变,远山剪影、飘落树叶粒子、阳光斑驳光晕、晨雾柔光
- **字体**：
  - 标题:`ZCOOL KuaiLe`(站酷快乐体)圆润童趣
  - 正文:`Noto Sans SC`思源黑体清晰易读
- **布局**：左侧固定导航栏(森林小径造型) + 右侧主内容区(玻璃卡片堆叠)
- **图标**：使用 lucide-react,搭配森林元素 emoji(🌲🌿🍃🦌🍄⭐)
- **按钮风格**：圆润大圆角(16px),毛玻璃质感,hover 时呈现露珠光泽
- **动效**：树叶飘落背景动画、卡片悬浮浮起、星愿币掉落弹跳、专注时进度环呼吸光晕

### 4.3 响应式

- **桌面优先**：1280px+ 宽屏布局,左侧导航 + 右侧内容区
- **平板适配**：768px+ 顶部导航,内容单列流式
- **移动端**：375px+ 底部 Tab 导航,卡片堆叠,触摸友好大按钮

### 4.4 3D 场景指引

本项目不使用 3D 场景,采用 CSS + SVG 实现森林氛围装饰:
- 多层 CSS 渐变模拟森林纵深
- SVG 树木剪影作为背景装饰
- CSS 动画实现树叶飘落、阳光闪烁
- 毛玻璃 backdrop-filter 实现景深感
